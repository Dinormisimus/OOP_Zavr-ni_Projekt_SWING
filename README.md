# OOP_Zavr-ni_Projekt_SWING

Opis projekta:

    - Java Swing RPG video igra
    - Igrač preuzima uloga jedne od klasa(Warrior, Rogue, Mage) i kreće u endless battle protiv nasumičnih neprijatelja
    - Svaka klasa ima svoje jedinstvene sposobnosti i statistike
    - Igrač može skupljati iskustvene bodove
    - Cilj aplikacije je testirati jednostavne battle mehanike i implementirati osnovne RPG elemente

Uputa za prve korisnika:

    !Important!!!
    1. U projektnom folderu napravite saves folder ( nisam znao kako napraviti gitignore samo za saveove )
